## 修正过的问题清单

| 原题 | 修正/说明 |
| --- | --- |
| 定义函数时，缺省函数的返回值类型，则函数的返回值类型为？ | 现代 C++ 中普通函数不能省略返回类型；旧 C 的隐式 `int` 规则不应作为现代答案。 |
| `char* p = "abcdefg...z"` 强转为 `int*` 后自增求值 | C++ 中字符串字面量应为 `const char[]`，强转和解引用涉及别名、对齐、端序，不能给可移植固定值。 |
| `bool less(float x, float y) { return *(int *)&x < *(int *)&y; }` | 违反严格别名规则，且不能正确处理 IEEE 754 负数、NaN、`-0` 等。 |
| `virtual()=0` 是什么意思 | 修正为纯虚函数声明 `virtual void f() = 0;`。 |
| `sizeof(A)` 固定求值 | 已说明依赖 32/64 位、ABI 和对齐规则，不能脱离平台给绝对答案。 |
| “可重入函数为什么一定是线程安全的” | 修正为可重入通常比线程安全要求更强，但仍要注意外部资源和调用环境。 |
| “聚合索引” | 统一按数据库常用术语“聚簇索引”解释。 |
| 指针强转计算题 | 已注明 32 位、小端、对齐等前提。 |
