## 各文件大纲

### 01_C-C++基础.md

- C/C++ 程序内存、内存碎片、虚拟内存
- 编码、浮点数、编译链接、进程加载
- C/C++ 互调、类型转换、函数指针、指针和引用
- const/static/extern "C"/inline/macro
- malloc/free、new/delete、智能指针
- C++11、lambda、模板、常见错题纠正

### 02_面向对象.md

- 面向对象三大特性
- 继承、访问控制、构造析构顺序
- 多态、虚函数表、虚继承
- 深浅拷贝、拷贝构造、赋值运算符
- 对象模型和 sizeof

### 03_STL.md

- STL 组成
- vector/list/deque/map/set/unordered_map
- 迭代器失效、erase、clear、swap
- 容器内存结构和适用场景

### 04_Linux与进程线程.md

- Linux 命令、文件系统、动态库、gdb、core dump
- 静态库/动态库、零拷贝、mmap
- 进程线程、IPC、同步、死锁、线程池、线程安全

### 05_网络.md

- OSI/TCP-IP 模型、URL 访问流程、socket 编程
- select/poll/epoll、LT/ET、Reactor/Proactor
- TCP/UDP、三次握手、四次挥手、TIME_WAIT
- HTTP/HTTPS、ARP、SSH、网络瓶颈优化

### 06_数据结构与算法.md

- 红黑树、Top K、哈希、一致性哈希、B/B+树、LRU
- 链表、栈、树遍历、排序、二分查找
- 海量数据、字符串匹配、LeetCode 题型
- 其它编程题思路

### 07_数据库与Redis.md

- MySQL 事务、隔离级别、锁、索引、查询优化
- 表连接、约束、视图、存储过程
- Redis 数据结构、持久化、线程模型、部署、主从哨兵、缓存问题

### 08_设计模式与项目.md

- 组合优于继承、单例、适配器、观察者
- 网盘项目
- 搜索引擎项目
- HTTP 项目

### 09_面试真题.md

- 头文件保护、include、namespace、define
- sizeof、声明定义、extern "C"、static
- 指针计算、模板、switch、内存问题、编译过程、英文翻译
